# Путеводитель

- [**Онлайн конструктор металлоконструкций**](https://github.com/lectorem-dev/Chgzabor) – веб-приложение для генерации и скачивания PDF-чертежей ворот с настройкой размеров, цвета, сечения труб и замков. Автоматическое масштабирование чертежей, готовые ТКП. Реализация в микросервисной архитектуре на Docker.
- [**Яндекс Алгоритмы**](https://github.com/lectorem-dev/Algorithms) – проходил Яндекс Тренеровки (2025г.), изучал алгоритмы (LeetCode, CodeRun).
-	[**MVP аналитического модуля маркетплейса**](https://github.com/lectorem-dev/analytic-module) – потоковая аналитика пользовательских событий через Kafka, ClickHouse, REST API и визуализация метрик в реальном времени. Реактивный фронтенд на React + Chart.js.
- [**Bloodhound**](https://github.com/lectorem-dev/Bloodhound) – эксперементировал с Apache Lucene поиск текста по PDF с поддержкой кириллицы через REST API. Индексация страниц, возврат топ-10 релевантных совпадений, локальное хранение файлов и Swagger-документация.
- [**gRPC-vs-HTTP2**](https://github.com/lectorem-dev/gRPC-vs-HTTP2) – учебный проект сравнения производительности потокового взаимодействия через gRPC и WebFlux HTTP/2. Микросервисы на Spring Boot, измерение latency, throughput, tail-perc и анализ удобства реализации.
- [**Weather Visualization**](https://github.com/lectorem-dev/WeatherAnalyzer) – микросервисное веб-приложение на Spring Boot + React. Визуализация погодных данных с генерацией реального времени, JWT-аутентификация, Docker-контейнеризация и микросервисная архитектура.
- [**Pet (Kafka, ClickHouse и UI)**](https://github.com/lectorem-dev/Pet) – демонстрация микросервисной архитектуры для обмена сообщениями и хранения данных в ClickHouse с графическим управлением Kafka и ClickHouse через UI.
- [**ViewerLDAP**](https://github.com/lectorem-dev/ViewerLDAP-v2) – Windows Forms клиент для просмотра корпоративной адресной книги через LDAP. Поиск, сортировка, пагинация и отображение текущего местоположения сотрудников.
- [**Курсовая работа: Управление банковскими счетами**](https://github.com/lectorem-dev/BeginningProgramming) – консольное приложение на C++ с хранением клиентов в CSV, интерактивным меню, контекстным поиском и WinAPI для оформления консоли.
- [**ATmega16Labs**](https://github.com/lectorem-dev/ATmega16Labs) – лабораторные работы и три калькулятора (LCD, LED, 7-segment) на микроконтроллере ATmega16. Ввод двоичных чисел через порт и базовые арифметические операции.
- [**Лабиринт для Nokia 3310**](https://github.com/lectorem-dev/Nokia3310-Atmega16) – текстовая игра на AVR Assembler с управлением через кнопки и отображением на дисплее PCD8544.
- [**ЖК-индикация (Nokia 3310, SPI, Assembler)**](https://github.com/lectorem-dev/ATmega16Labs) – управление графическим LCD дисплеем PCD8544 на ATmega16 через SPI.
